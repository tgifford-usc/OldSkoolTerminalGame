## Run the Terminal Game in your browser

[![Open in Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/tgifford-usc/OldSkoolTerminalGame)

When the environment loads:

1. Edit `src/TerminalCartridge.cpp`
2. Edit `include/TerminalCartridge.h`
3. Run **Terminal → Run Task → ▶ Run Terminal Game**
